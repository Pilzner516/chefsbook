## Overview

Batch-import all recipes from *The Food Lab: Better Home Cooking Through Science*
by J. Kenji López-Alt that are published on Serious Eats (seriouseats.com) into
the authenticated user's ChefsBook recipe library.

The task runs in two phases:

**Phase 1 — Discovery** *(parallelise across all 9 chapters using ultrawork)*
Use web search via the Anthropic API to compile a full list of Serious Eats
recipe URLs authored by Kenji López-Alt, organised by the book's 9 chapters.
All 9 chapter searches run in parallel. Output the complete deduplicated list to
`scripts/food-lab-urls.json` before importing anything. This file is the
idempotent source of truth for Phase 2.

**Phase 2 — Import** *(sequential, rate-limited)*
Call the existing `POST http://localhost:3000/api/import/url` endpoint for each
URL. This route already handles the Cloudflare block on seriouseats.com via the
ScrapingBee fallback. Do not bypass or reimplement the import pipeline — use it
exactly as-is.

Idempotency rules:
- If `scripts/food-lab-urls.json` already exists, skip Phase 1 entirely.
- If a URL returns `{ duplicate: true }`, log as SKIPPED and continue.
- Never crash the script on a single URL failure.

Tag every imported recipe with: `food-lab`, `kenji-lopez-alt`.

## Phase 2 — Batch Import (sequential, 1 request per 3 seconds)

Read `scripts/food-lab-urls.json`. For each recipe URL:

**Request:**
```
POST http://localhost:3000/api/import/url
Authorization: Bearer <CHEFSBOOK_LIBRARY_TOKEN>
Content-Type: application/json

{
  "url": "https://www.seriouseats.com/...",
  "userLanguage": "en",
  "tags": ["food-lab", "kenji-lopez-alt"]
}
```

**Response handling:**
- `{ recipe }` → log as `IMPORTED`, record title and id
- `{ duplicate: true }` → log as `SKIPPED`
- `{ error }` or non-200 → log as `FAILED`, record URL and error message.
  For ScrapingBee 503/429: wait 10 seconds and retry once before marking FAILED.

**Rate limit:** 1 request per 3 seconds. ScrapingBee is a paid API — do not
hammer it.

**Append to `scripts/food-lab-import-log.json` after each request:**
```json
{ "url": "...", "status": "imported|skipped|failed", "title": "...", "recipeId": "...", "error": "...", "timestamp": "..." }
```

**Live stdout progress:**
```
[12/147] IMPORTED  — The Best Crispy Roast Potatoes Ever
[13/147] SKIPPED   — Foolproof Pan Pizza (already in library)
[14/147] FAILED    — https://... (ScrapingBee 503, retried once)
```

**Final summary:**
```
Phase 2 complete: 147 URLs processed
  Imported:  134
  Skipped:     8
  Failed:      5
Log saved to scripts/food-lab-import-log.json
```
